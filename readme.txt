创建图形化工具（Java Swing）的一套框架，以及部分工具。

由框架本身将参数/动作等概念抽象出来，提供图形化界面。

工具则只需要提供参数配置信息（参数名称/类型/枚举值等），实现具体功能（且可以通过UserInterface接口输出信息）。

通过这样的框架架构，可以将原本适用于命令行的工具简单改造成图形化界面工具（命令行参数调整为Parameter参数，命令行输出变成UserInterface输出），一下子高大上数倍。

预置的工具包括：
1. 文件中文编码批量修正
2. 身份证号自动修正（最后校验位）
3. 随机号码生成（社会统一信用证号，组织机构证号，行政区划号，工商营业执照号）
4. 一个WEB工程按文件清单增量打包的工具
