创建图形化工具（Java Swing）的一套框架，以及部分工具。

由框架本身将参数/动作等概念抽象出来，提供图形化界面。

工具则只需要提供参数配置信息（参数名称/类型/枚举值等），实现具体功能（且可以通过UserInterface接口输出信息）。

通过这样的框架架构，可以将原本适用于命令行的工具简单改造成图形化界面工具（命令行参数调整为Parameter参数，命令行输出变成UserInterface输出），一下子高大上数倍。

注：新增了启动参数 -Dscale=1.0 ，可以设置界面缩放比例，如设置为1.5，则窗口和字体大小为1.5倍等。

预置的一些工具包括：

0000_script: 自定义脚本执行
	JSR233的一个简单包装，可以执行Groovy或Javascript（JDK8的Nashorn引擎，将来的JDK可能不支持了）。
	Groovy脚本可以当简单的脚本计算器来使用

0001_packager: 打包工具：增量打包
	一个Java WEB工程/普通Java工厂按文件清单增量打包的工具
	
0101_textFileEndoding: 文本处理：中文编码转换
	对某个目录下文件进行批量中文编码修正
	
0102_textFileLineEnding: 文本处理：换行符转换
	对某个目录下的文件进行批量换行符修正
	
0103_textFileRemoveBom: 文本处理：去除BOM
	对某个目录下的文件进行批量去除BOM
	
0104_textFileLineCount: 文本处理：统计行数
	对某个目录下的文件进行行数统计

0105_combineTextFile：文件处理：文本文件合并
	将一个目录下所有文件内容合并，适用于如一堆sql脚本整合到一起之类的场景
	
0106_splitJavaAndResource：文件处理：分离java/resource文件
	用于传统项目转Maven项目时，将单一源码目录中的java文件和其他文件分离开

0201_escape: 转义：URL/XML/JS文本转义
	进行URL/XML/JS的转义/反转义

0301_idNumber: 证件号：身份证号修正
	将输入的身份证号自动修正符合校验规则
	
0302_randomCode: 证件号：随机证件号生成
	生成一些符合校验规则的随机证件号，如社会统一信用证号，组织机构证号，工商营业执照号等

0401_digest: 摘要：摘要值计算
	计算MD5/SHA1等摘要值

0402_digestCrack: 摘要：摘要密码暴力反算
	暴力反算MD5/SHA-1摘要后的密码，获取原密码（注：从性能上只能破解比较短的密码）
	
0501_qrcodereader: 二维码：二维码解析
	从图片解析二维码获取文本值
	
0502_qrcodewriter: 二维码：二维码生成
	将文本转换为二维码图片

0601_imageCapture: 一个简单的截图工具
	实现了简单的屏幕区域截图功能

0602_ocr: 借助在线OCR API服务进行图片文字识别
	目前支持百度云的API

0701_dbExport: 数据导出
	简单的数据库表数据导出功能，与导入功能结合使用，可以跨不同类型数据库进行处理

0702_dbImport: 数据导入
	结合数据导出功能，将导出的文件再导入到数据库中
